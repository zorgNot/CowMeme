# CowMeme

*Meme's for Cowman's favorite guildies*

A World of Warcraft TBC Anniversary addon for guild shenanigans.

## Features

- **CopyPasta** — fire pre-written copypastas at specific guildies, by hand or automatically. A gamba monitor watches CrossGambling games and drops a pasta on whoever the host says "owes".
- **Fast and Clean (FnC)** — a death tracker for runs. Counts party/raid deaths, calls out milestones and first blood, reports cause of death, and can announce to party/raid/guild or a numbered channel.
- **Panel** — a small movable on-screen panel for images and text: copypasta art, the gamba host's latest "owes" line, and live top/bottom rolls for the current game.
- **Sync** — when multiple guildies run the addon, a leader is elected so only one client actually speaks in chat.

## Commands

- `/cm` — main menu, status, and settings
- `/cm help` — list all commands (`/cm help <command>` for details)
- `/fnc` — Fast and Clean death tracker
- `/cp` — CopyPasta

## Installation

Drop the `CowMeme` folder into `World of Warcraft\_anniversary_\Interface\AddOns\` and reload.
